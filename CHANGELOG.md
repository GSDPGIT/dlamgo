# 更新日志

本文件用于记录 DLAMGO 的关键版本变更。

当前仓库已经具备自动发布工作流：

- 推送 `v*` 标签时，会自动构建发布附件
- Release 附件中会自动包含版本说明和本次变更摘要

因此本文件建议作为“人工整理后的长期变更记录”，而不是每次提交都手动维护的流水账。
